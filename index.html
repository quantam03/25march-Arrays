Arrays

  • Map:is a js method which is used to transform for an array i.e which will iterate through each and every element

  • Filter:this method will check the array and filter accordingly and return a new array.

  • Reduce:this method reduces an array of values down just to one avlue .to get the o/p value,it runs a reducer function on each element of the array

  • Index:return the value of the element in that index
    ex. const arr = [2,3,4]; console.log(arr.indexof(3))
    o/p 1

  • Push:Push is array method used to append data into the end of the array.

  • Pop:Pop is a array method which is used to remove the last element of the array.

  • Unshift:Unshift is a array method to add item in the beginning of the array.

  • Shift:Shift is array method which is used to remove the first element in the array.

  • Some:if even one condition is true it returns true

  • Every:if every condition is true it returns true

  • Slice:arr.slice(starting value,ending value)
    it will not modify the array
    it will try to remove the elemnts from the array
    includes starting value but not ending value

  • Splice:arr.splice(starting value,ending value)
    it will not modify the array
    it will try to remove the elemnts from the array
    includes starting value and ending value also.
